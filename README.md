# spark-playground
spark-playground is a repository with some pocs around apache spark

## Apache Spark 4.2

- [spark-4.2-scala3-fun](spark-4.2-scala3-fun) — Spark 4.2.0 from Scala 3.7.4, podman, React/TypeScript 7 UI.
- [spark-4.2-python3-fun](spark-4.2-python3-fun) — Spark 4.2.0 from Python 3.12 (PySpark), podman, React/TypeScript 7 UI.

Both exercise the new 4.2 features — metric views, `NEAREST BY` vector search, sketches, and native geospatial types — and visualise the results in a typed UI fed by a single JSON file.
